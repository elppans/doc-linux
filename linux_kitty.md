# Guia Rápido de Atalhos do Kitty Terminal e Sistema

Este mini tutorial reúne os atalhos de teclado e Mouse para o gerenciamento de janelas, abas, fontes, navegação no Kitty Terminal e manipulação de janelas no ambiente gráfico.

![alt text](<img/linux_kitty.png>)
## Manipulação de Janelas (Interface Gráfica)

| Ação | Atalho |
| --- | --- |
| Mover janela | `Super` + `Botão esquerdo do Mouse pressionado` |
| Menu com atalhos de janela (Maximizar, Mover, Redimensionar, etc.) | `Super` + `Botão direito do Mouse` |

## Gerenciamento de Abas e Janelas no Kitty

| Ação | Atalho |
| --- | --- |
| Abrir nova aba | `Ctrl` + `Shift` + `T` |
| Fechar aba atual | `Ctrl` + `Shift` + `W` |
| Abrir nova janela | `Ctrl` + `Shift` + `N` |
| Fechar janela atual | `Ctrl` + `Shift` + `Q` |

## Área de Transferência (Clipboard)

| Ação | Atalho |
| --- | --- |
| Copiar conteúdo selecionado | `Ctrl` + `Shift` + `C` |
| Colar conteúdo | `Ctrl` + `Shift` + `V` |

## Controle de Fonte (Zoom)

| Ação | Atalho |
| --- | --- |
| Aumentar o tamanho do texto | `Ctrl` + `Shift` + `+` |
| Diminuir o tamanho do texto | `Ctrl` + `Shift` + `-` |

## Navegação e Histórico (Scrollback)

| Ação | Atalho |
| --- | --- |
| Rolar para o topo (Primeira linha) | `Ctrl` + `Shift` + `Page Up` ou `Home` |
| Rolar para o final (Última linha) | `Ctrl` + `Shift` + `Page Down` ou `End` |
| Rolar para cima (Linha por linha) | `Ctrl` + `Shift` + `Seta para Cima` |
| Rolar para baixo (Linha por linha) | `Ctrl` + `Shift` + `Seta para Baixo` |

## Modos de Visualização e Ajuda

| Ação | Atalho |
| --- | --- |
| Mostrar o Help no navegador | `Ctrl` + `Shift` + `F1` |
| Consultar mapa de atalhos completo | `Ctrl` + `Shift` + `F3` |
| Alternar Full Screen (Tela Cheia Opaca) | `Ctrl` + `Shift` + `F11` |
| Alternar Full Screen (Mantendo Transparência) | `Ctrl` + `Shift` + `F10` |
